Class: BackgroundObject

BackgroundObject(imagePath, x)

Represents a background object in the game, such as scenery or static elements that contribute to the visual background. Inherits from MovableObject.

Constructor

new BackgroundObject(imagePath, x)

Initializes a new background object with a specified image and x-coordinate. Positions the object vertically at ground level based on its height.
Parameters:
Name Type Description
imagePath string The path to the image representing the background object.
x number The x-coordinate for the background object's position.
