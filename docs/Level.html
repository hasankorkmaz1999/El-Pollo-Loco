Class: Level

Level(enemies, clouds, backgroundObjects, bottles, coins)

The Level class defines the environment and objects within a game level, including enemies, clouds, background elements, bottles, and coins.

Constructor

new Level(enemies, clouds, backgroundObjects, bottles, coins)

Instantiates a new Level with specified enemies, clouds, background objects, bottles, and coins.
